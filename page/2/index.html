异常记录簿

Resolved [org.springframework.web.bind.MissingServletRequestParameterException: Required request parameter email for method parameter type String is not present]

JavaMailSender的send方法报错Local address contains illegal character in string

问题分析

本来以为是接收到的参数问题，结果让前端修改了格式还是报错
最后在controller打印信息发现是接收参数方式错了
前端传的email是JSON对象，类似{“email”:”xxx@qq.com“}，而我直接使用
@RequestParam(“email”) String email接收，自然就报错了

解决方案

将接收参数的方式改为@RequestBody Map<String,Object> map
使用map.get(“email”)获取值即可
