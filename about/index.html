项目经历

项目一：疫情时空可视化分析平台

职责：Java开发

项目简介：根据初筛阳性和无症状感染者的时空轨迹数据、中高风险区的位置数据，基于电子地图，实现数据的地图显示、时空交集分析、影响范围分析、安全出行路线推荐等功能

技术栈：SpringBoot + MyBatis + MySQL + JavaScript + Vue + GIS

工作内容：

  • 收集疫情感染者的时空轨迹数据和中高风险区的位置数据，提取并存储数据中的有效信息。

  • 基于 GIS 技术和 JavaScript 编写地图可视化程序，实现感染者轨迹数据的地图显示，实现用户输入自己的轨迹数据，与感染者时空轨迹进行重叠判断

  • 根据感染者的时空轨迹周边 30 米和中高风险区的人口密度，基于缓冲区分析，实现潜在扩散风险的判断。

  • 根据用户的起止点以及中高风险区的位置，基于 Dijkstra 算法以及地图的网格表示法，实现避开中高风险区的行进路线规划。
